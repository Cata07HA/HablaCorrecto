# HablaCorrecto

**HablaCorrecto** es un juego educativo en Python diseñado para ayudar a los estudiantes a mejorar su escritura y pronunciación de palabras en español a ingles .  
El objetivo es practicar vocabulario de distintos niveles de dificultad y reforzar la correcta ortografía.

## Para qué sirve?
- Desarrollar habilidades de pronunciación y escritura.  
- Aprender vocabulario básico, intermedio y avanzado.  
- Practicar de manera interactiva y divertida.  

El juego presenta palabras organizadas por niveles de dificultad:

**Nivel fácil:**  
- gato      cat
- perro     dog
- manzana   apple
- leche     milk
- sol       sun

**Nivel medio:**  
- platano      banana  
- escuela      school
- amigo        friend
- ventana      window
- amarillo     yellow

**Nivel difícil:**  
- tecnología      technology
- universidad     university
- información     information
- pronunciación   pronunciation
- imaginación     imagination
